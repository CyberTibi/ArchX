# ArchX Linux

ArchX is a minimalist Arch Linux–based distribution designed to provide a simple installation process and a clean user experience.  
Its goal is to deliver the flexibility of Arch Linux in a user-friendly way, with fast and easy installation through the Calamares installer.

## Key Features

- Arch Linux base with rolling release updates  
- User-friendly **Xfce4** desktop environment  
- Calamares installer with ArchX custom slideshow  
- Minimalist, modern visual design  
- Minimal set of preinstalled applications, allowing users to customize their system freely  
- Open source, community-driven development  

## Preinstalled Applications

- **Thunar** - file manager  
- **xfce4-terminal** - with Zsh + Powerlevel10k theme  
- **Firefox** - web browser  
- **Evince** - document viewer  
- **GParted** – partitioning and disk management tool  
- **Pamac** - software manager  
- **NetworkManager + iwd + dnsmasq** - for network management  
- **yay** - AUR helper  
- Bluetooth support  
- Printer support  

## Installation

1. Download the ISO file.  
2. Write it to a USB drive (using `balenaEtcher`, `Rufus`, or `dd`).  
3. Boot from the USB drive.  
4. Launch the Calamares installer.  
5. Follow the six-step slideshow to install the system.  

## System Requirements

- 64-bit processor  
- Minimum 2 GB RAM  
- 20 GB free disk space  
- USB or DVD installation media
